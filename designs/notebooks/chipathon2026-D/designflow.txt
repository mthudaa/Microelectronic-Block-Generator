│  │  └─────────────────────────────────────────────────────────────┘ │   │
│  │                              │                                    │   │
│  │                              ▼                                    │   │
│  │  ┌─────────────────────────────────────────────────────────────┐ │   │
│  │  │  PATHFINDER OUTER LOOP  (max 10 iterations)                  │ │   │
│  │  │  penalty_threshold = max(0, 100 - iter × 15)                 │ │   │
│  │  │  ┌─────────────────────────────────────────────────────────┐ │ │   │
│  │  │  │  INNER ROUTING LOOP  (max 10 rip-up retries)            │ │ │   │
│  │  │  │                                                          │ │ │   │
│  │  │  │  For each edge:                                          │ │ │   │
│  │  │  │    1. Sort port combos by distance (shortest first)      │ │ │   │

│  │  │  │    3. Try patterns in order:                             │ │ │   │

│  │  │  │       │ L-ROUTE  → one turn (3 vias)                │    │ │ │   │
│  │  │  │       │ Z-ROUTE  → midpoint sweep (4 vias)          │    │ │ │   │
│  │  │  │       │ U-ROUTE  → detour around obstacles (5 vias) │    │ │ │   │
│  │  │  │       └─────────────────────────────────────────────┘    │ │ │   │
│  │  │  │                                                          │ │ │   │
│  │  │  │  Layer priority: M3 → M5 fallback                        │ │ │   │
│  │  │  │  Trace width: 0.5µm uniform                              │ │ │   │
│  │  │  │  Via offset: w/2 opposite port direction                 │ │ │   │
│  │  │  │  Trace extend: w/2 at both ends (overlap via)            │ │ │   │

║                        KEY ALGORITHMS                                    ║
╠══════════════════════════════════════════════════════════════════════════╣
║                                                                          ║
║  PATHFINDER NCR (Negotiated Congestion Routing)                          ║
║  ─────────────────────────────────────────                               ║
║  Reference: "Analog IC Routing Techniques — An Extensive Review"         ║
║  IEEE Access, 2023                                                        ║
║  • Iterative rip-up & reroute with congestion history                    ║
║  • Penalty threshold decreases each iteration (100 → 0)                  ║
║  • Device geometry as permanent hard blocks                              ║
║                                                                          ║
║  ALIGN-INSPIRED PLACEMENT                                                 ║
║  ─────────────────────────                                               ║
║  Reference: "ALIGN: A System for Automating Analog Layout"               ║
║  IEEE Design & Test, 2021                                                 ║
║  • PMOS always top row, NMOS bottom row                                  ║
║  • Connectivity-driven column ordering                                   ║
║  • Bipartite PMOS-NMOS edge weighting                                    ║
║  • Center alignment for symmetry                                         ║
║                                                                          ║
║  MST DECOMPOSITION                                                        ║
║  ─────────────────                                                       ║
║  • Prim's algorithm for minimum spanning tree                            ║
║  • Multi-pin nets → optimal edge pairs                                   ║
║                                                                          ║
║  PATTERN-BASED ROUTING                                                    ║
║  ───────────────────────                                                  ║
║  • I-route: straight single-layer (fewest vias)                          ║
║  • L-route: one corner turn                                              ║
║  • Z-route: midpoint sweep with dynamic clearance                        ║
║  • U-route: detour around obstacles                                      ║
║  • Layer: M3 (horizontal) → M4 (vertical) → M5 (overpass)                ║
║                                                                          ║
